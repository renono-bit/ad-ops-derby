# 【理不尽】Web広告運用ダービー（マルチプレイ版）

Web広告運用のプランに持ちポイントを賭けて、KPI達成率レースの結果で払い戻しを競う、**1〜4人対戦**のダービー風パーティーゲームです。

1台の画面をみんなで囲んでも、Web会議の画面共有でも遊べます。

## 遊び方

1. 参加人数（1〜4人）と名前を登録
2. 全員 **1,000pt** でスタート
3. 毎レース、賭け金（100 / 300 / 500pt）を選んで運用プランに投票
4. レース中はクライアント横やり・媒体事故・奇跡などの理不尽イベントが発生
5. **1着的中でオッズ×賭け金**を獲得、2着なら賭け金返還
6. 全5案件終了時点で持ちポイントが最も多い人が優勝

ポイントが尽きても「来期予算の前借り」で自動補填されるので、最後まで遊べます。

## 特徴

- 🎮 1〜4人のホットシート対戦（順番に馬券を購入）
- 🎵 Web Audio APIによるチップチューンBGM・ファンファーレ・SE（外部音源ファイル不要）
- 🕹️ ドット系ゲームフォント（DotGothic16 / Press Start 2P）
- ⚡ 理不尽イベントは案件固有+汎用プールからランダム抽選（毎回展開が変わる）
- 🎊 最終結果画面の紙吹雪と𝕏シェアボタン

## ローカル確認

`index.html` をブラウザで直接開くか、任意の静的サーバーで配信します。

```powershell
py -3.12 -m http.server 8000
```

その後 `http://127.0.0.1:8000/` を開きます。
